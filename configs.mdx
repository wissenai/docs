This refers to the configurations that are defined which impacts a given `final_result`.


1. A declared generation
For a generation to be valid, it requires instantiating
- gen_type
- input_model
- model, provider, and model configurations
- tools
- instructions
    - task prompt
    - tool prompt

optional
- output_model
- gen_category 
- total_token_limit
- max_turns

